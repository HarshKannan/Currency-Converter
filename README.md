## Overview

Creating a currency converter application in Java, with the added functionality of a user-friendly graphical interface using Java Swing in NetBeans, offers a powerful tool for seamless currency conversion. This project leverages the versatility and scalability of Java to provide users with a straightforward means of converting between different currencies based on user-defined exchange rates. With this self-contained solution, users can effortlessly perform currency conversions, making it a valuable addition to both personal finance management and educational contexts. Whether you're a student learning Java or a financial enthusiast seeking a practical tool, this project demonstrates the power of Java in building intuitive and efficient software.

The primary objective of this project is to create a user-friendly currency converter application  that helps user to convert different rates.
